# DEVPROFILE - Peramanathan Sathyamoorthy Portfolio Website

## Project Purpose
A modern, full-stack web application showcasing Peramanathan Sathyamoorthy's professional portfolio as a Senior Software Engineer, featuring AI-powered AMA chatbot, interactive document viewing, and automated PDF CV generation.

## Core Requirements
- **Professional Portfolio Display**: Showcase engineering experience, skills, projects, and achievements
- **Interactive CV System**: Dynamic PDF generation with professional styling and cross-platform compatibility
- **AI-Powered Features**: AI SDK v6 + Workflow DevKit with semantic CV embeddings and tool-based Q&A
- **Content Management**: Dynamic multi-page content hub for blogs, articles, and resources
- **Document Viewer**: Interactive PDF/image viewing with full browser integration
- **Responsive Design**: Mobile-first approach optimized for all device sizes
- **Performance Optimized**: Fast loading with Next.js 15 App Router and modern web standards
- **SEO Optimized**: Server-side rendering with proper meta tags and structured data
- **Accessibility**: WCAG compliant with keyboard navigation and screen reader support

## Technical Vision
Build a 2025-compliant web application using the latest web technologies and best practices, demonstrating modern full-stack development capabilities while maintaining production-ready deployment flexibility across Vercel, Netlify, and AWS platforms.

## Scope Boundaries
- **In Scope**: Portfolio content, CV generation, AI AMA, document viewing, responsive design
- **Out of Scope**: E-commerce functionality, third-party integrations beyond AI APIs, complex database systems
- **Constraints**: Must use modern web technologies (Next.js 15, React 19), maintain TypeScript strict mode, ensure cross-platform deployment compatibility

## Success Metrics
- Performance: Build time < 5s, Core Web Vitals compliant
- Accessibility: WCAG AA compliance across all components
- SEO: Full meta tag implementation and structured data
- User Experience: Mobile-first responsive design with intuitive navigation
- AI Integration: 95%+ accurate responses with semantic CV search and tool specialization
- Deployment: Zero-config deployment capability across major platforms
